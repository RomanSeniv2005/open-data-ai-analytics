# Аналіз ринку електромобілів в Україні (Open Data AI Analytics)

**Мета проєкту:** Дослідити динаміку реєстрацій електрокарів (EV) в Україні станом на 2026 рік, виявити ключові тренди на ринку (зокрема вплив імпорту та розвиток регіональної інфраструктури) та підготувати очищений датасет для побудови предиктивних моделей машинного навчання.

**Джерело відкритих даних:** [Відомості про транспортні засоби та їх власників (data.gov.ua)](https://data.gov.ua/dataset/120cb6b6-d242-49af-ba3a-449e7bdfa6e4)

**3 гіпотези для аналізу:**
1. У 2026 році частка нових реєстрацій електромобілів становить понад 25% від загальної кількості всіх вперше зареєстрованих легкових авто в Україні, що підтверджує сталий тренд відмови від двигунів внутрішнього згоряння.
2. У структурі імпорту електромобілів частка авто китайського виробництва (наприклад, BYD, Honda M-NV, Volkswagen ID.4 з Китаю) остаточно випередила імпорт вживаних електрокарів зі США завдяки більшій ціновій доступності.
3. Частка реєстрацій електромобілів у західних регіонах впритул наблизилась до показників столиці завдяки активній розбудові мережі швидкісних зарядних станцій протягом останніх років.Цей рядок додано з гілки А
