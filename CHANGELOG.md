# Changelog

Всі помітні зміни в цьому проєкті будуть документуватися в цьому файлі.

## [v0.1.0] - 2026-02-26

### Додано
- Базову структуру Data Science проєкту (папки `data`, `notebooks`, `src`, `reports`).
- Файл `.gitignore` для виключення системних файлів, віртуальних оточень та сирих даних.
- `README.md` з описом проєкту, джерелом даних (data.gov.ua) та гіпотезами щодо ринку електромобілів (EV) у 2026 році.
- Скрипт `src/data_load.py` для завантаження даних.
- Скрипт `src/data_quality.py` для перевірки якості даних (пропуски, дублікати).
- Скрипт `src/data_research.py` для аналізу трендів та побудови моделі прогнозування попиту.
- Скрипт `src/visualization.py` для генерації графіків динаміки реєстрацій електромобілів.

### Змінено
- Успішно зімітовано та розв'язано merge-конфлікт при одночасній зміні файлу README.md у різних гілках.
